## 🎯 使用方法

### 1. 基本使用

```python
from src.download.multi_source_download import create_multi_source_downloader

# 创建下载器
downloader = create_multi_source_downloader(
    email="your-email@example.com",  # 替换为你的邮箱
    output_dir="pdfs/all"
)

# 下载单篇
result = downloader.download("10.1016/j.energy.2023.128788")
print(f"成功: {result.success}, 来源: {result.source.value}")
```

### 2. 配置邮箱

Unpaywall API 需要一个有效的邮箱地址（免费获取）：

**选项 A**: 使用你自己的邮箱
```python
downloader = MultiSourceDownloader(email="myemail@example.com")
```

**选项 B**: 从配置文件读取
```yaml
# config/api_keys.yaml
unpaywall:
  email: "your-email@example.com"
```

```python
import yaml
with open("config/api_keys.yaml") as f:
    config = yaml.safe_load(f)
    email = config["unpaywall"]["email"]

downloader = MultiSourceDownloader(email=email)
```

### 3. 批量下载

```python
doi_list = [
    "10.1016/j.energy.2023.128788",
    "10.1038/nature12373",
    # ... 更多 DOI
]

results = downloader.download_batch(
    doi_list,
    delay=2.0,  # 每次下载间隔 2 秒
    progress_callback=lambda cur, total, res: print(f"{cur}/{total}: {res.source.value}")
)

# 查看统计
stats = downloader.get_stats()
print(f"总成功率: {stats['total_success_rate']:.1f}%")
```

## 🔧 常见问题

### Q1: Unpaywall 返回 HTTP 422 错误？

**原因**: 邮箱地址无效或格式错误

**解决**: 使用有效的邮箱地址
```python
# 错误示例
downloader = MultiSourceDownloader(email="test")

# 正确示例
downloader = MultiSourceDownloader(email="user@example.com")
```

### Q2: Unpaywall 找不到论文？

**原因**: 该论文没有开放获取版本

**解决**: 系统会自动尝试 SciHub，无需手动处理

### Q3: SciHub 全部返回 403？

**原因**: SciHub 镜像站被封锁或不稳定

**解决**:
1. 确保代理正常运行
2. 更新镜像站列表
3. 使用较老的论文（2020-2021年）

### Q4: 如何只使用 Unpaywall（不使用 SciHub）？

```python
downloader = MultiSourceDownloader(
    email="your-email@example.com",
    proxy_manager=None  # 不提供代理管理器
)

# 下载时禁用 SciHub
result = downloader.download(doi, try_scihub=False)
```

## 📝 配置示例

### config/api_keys.yaml

```yaml
# Elsevier API
elsevier:
  api_key: "your_elsevier_api_key"

# Unpaywall 邮箱（免费获取）
unpaywall:
  email: "your-email@example.com"

# GLM API（用于关键词翻译）
glm:
  api_key: "your_glm_api_key"
```

### 使用配置文件

```python
import yaml

with open("config/api_keys.yaml") as f:
    config = yaml.safe_load(f)

workflow = Page1Workflow(
    project_dir="data/projects/wind_aero",
    api_key=config["elsevier"]["api_key"]
)
```
