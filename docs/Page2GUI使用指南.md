# Page 2 GUI 使用指南

**功能**: AI 深度分析（双层并发 + DeepSeek 无限并发）

---

## 📋 使用流程

### 步骤 1: 选择 Markdown 目录

打开 Page 2 后，首先需要选择要分析的 Markdown 文献目录。

#### 方式 A: 自动检测（推荐）

1. 点击「🔍 自动检测」按钮
2. 系统会自动扫描并显示所有包含 Markdown 论文的目录
3. 检测结果示例：
   ```
   📁 markdown/all (默认输出，611 篇)
   📁 markdown/categories/Aerodynamic Optimization (45 篇)
   📁 markdown/categories/Wind Turbine Control (32 篇)
   ```
4. 选择要分析的目录，点击「确定」

#### 方式 B: 手动浏览

1. 点击「📁 浏览...」按钮
2. 在文件对话框中选择 Markdown 目录
3. 系统会显示选中的目录路径

### 步骤 2: 配置并发参数

根据您的硬件配置和 API 限制，调整并发参数：

| 参数 | 默认值 | 说明 | 推荐范围 |
|------|-------|------|---------|
| 外层并发 | 10 | 同时处理的论文数量 | 5-20 |
| 内层并发 | 5 | 每篇论文的分析器数量 | 3-5 |

**理论最大并发**: 外层并发 × 内层并发 = 10 × 5 = 50 个同时请求

### 步骤 3: 开始分析

1. 确认目录和参数设置正确
2. 点击「▶️ 开始分析」按钮
3. 系统开始自动分析，您可以：
   - 查看总体进度
   - 查看当前论文信息
   - 查看 5 个分析器的实时状态
   - 查看日志输出

### 步骤 4: 监控分析进度

**进度显示**:

### 步骤 5: 暂停/继续/停止

- **暂停**: 点击「⏸️ 暂停」按钮，分析会暂停
- **继续**: 暂停后，按钮变为「▶️ 继续」，点击可继续分析
- **停止**: 点击「⏹️ 停止」按钮，完全停止分析

## ⚙️ 常见问题

### Q1: 如何切换到不同的目录？

点击「🔍 自动检测」按钮，选择其他目录即可。

### Q2: 可以只分析部分论文吗？

可以！修改「外层并发」参数为较小的值（如 5），系统会优先分析排名靠前的论文。

### Q3: 分析失败了怎么办？

1. 查看日志输出，找到错误信息
2. 检查 DeepSeek API 密钥是否正确配置
3. 检查网络连接
4. 点击「▶️ 开始分析」重新开始（已完成的会自动跳过）

### Q4: 为什么有些分析器显示「失败」？

可能原因：
- API 请求超时
- 论文内容格式不正确
- Markdown 文件损坏

解决方法：查看日志，根据错误信息修复后重新分析。

### Q5: 如何加快分析速度？

1. 增加外层并发数（同时处理更多论文）
2. 增加内层并发数（每篇论文使用更多分析器）
3. 确保网络连接稳定
4. 使用 DeepSeek API（无限并发）
