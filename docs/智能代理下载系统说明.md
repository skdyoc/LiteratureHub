# 智能代理下载系统 - 实现说明

## 📋 完成的工作

### 1. ✅ ProxyManager 类（智能代理管理器）

**文件**: `src/download/proxy_manager.py`

**核心功能**:
- 从 YAML 配置或默认列表加载代理
- **并发延迟测试**（TCP + HTTP 双重测试）
- 智能选择最低延迟代理
- 代理统计跟踪（成功率、平均延迟）

**使用示例**:
```python
from src.download.proxy_manager import create_default_proxy_manager

# 创建默认代理管理器（Mihomo）
manager = create_default_proxy_manager()

# 选择最佳代理
best_proxy = manager.select_best_proxy()
print(f"最佳代理: {best_proxy.name}, 延迟: {best_proxy.latency:.0f}ms")
```

**关键特性**:
- `test_proxy_latency()`: TCP 连接测试 + HTTP 请求测试
- `select_best_proxy()`: 并发测试所有代理（ThreadPoolExecutor）
- `record_success()/record_failure()`: 跟踪代理成功率

### 2. ✅ SciHubDownloader 重构

**文件**: `src/download/scihub_downloader.py`

**改进内容**:
1. 集成 ProxyManager 智能代理选择
2. **每 N 篇自动切换代理**（proxy_switch_interval=10）
3. **失败时立即切换代理并重试**（max_retries=3）
4. 添加 **lxml HTML 解析器**增强 PDF 链接提取

**新方法**:
- `_update_session_proxy()`: 动态更新会话代理
- `_download_count`: 下载计数器（用于代理切换）

**PDF 链接提取优先级**:
1. **lxml + XPath 解析**（最可靠）
2. BeautifulSoup (embed/iframe 标签)
3. 正则表达式匹配（最后备选）

## 🔧 使用方式

### 方式1: 使用 ProxyManager（推荐）

```python
from src.download.proxy_manager import create_default_proxy_manager
from src.download.scihub_downloader import SciHubDownloader

# 创建代理管理器
proxy_manager = create_default_proxy_manager()

# 创建下载器
downloader = SciHubDownloader(
    proxy_manager=proxy_manager,
    output_dir="pdfs/all",
    proxy_switch_interval=10  # 每10篇切换一次代理
)

# 下载文献
result = downloader.download("10.1016/j.energy.2023.128788", max_retries=3)
```

### 方式2: 从 YAML 配置加载代理

```python
from src.download.proxy_manager import ProxyManager
from src.download.scihub_downloader import SciHubDownloader

# 从 Clash 配置加载
proxy_manager = ProxyManager.from_yaml("C:/Users/19874/.config/clash/profiles/config.yml")

downloader = SciHubDownloader(proxy_manager=proxy_manager)
```
