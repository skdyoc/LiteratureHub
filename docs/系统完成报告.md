## 🎯 核心功能

### 1. 图形用户界面 (GUI)
- **文件**: [scripts/page1_gui.py](scripts/page1_gui.py)
- **功能**: 完整的文献管理 GUI
- **特点**:
  - 中文界面
  - 实时进度显示
  - 日志输出
  - 文献列表和详情查看

### 2. 一键启动器
- **文件**: [launch_gui.py](launch_gui.py)
- **功能**: 自动检查依赖并启动 GUI
- **批处理**: [启动GUI.bat](启动GUI.bat) (Windows 双击启动)

### 3. 多源下载系统
- **文件**: [src/download/multi_source_downloader.py](src/download/multi_source_downloader.py)
- **策略**: Unpaywall (合法) 优先 + SciHub 备用
- **特点**:
  - 自动 VPN 检测和启动
  - 智能代理选择

### 4. 关键词翻译 Agent
- **文件**: [src/search/keyword_translator.py](src/search/keyword_translator.py)
- **功能**: 中文关键词自动翻译为英文
- **支持**: GLM-4.7-flash API

### 5. 智能代理管理器
- **文件**: [src/download/proxy_manager.py](src/download/proxy_manager.py)
- **功能**: 并发测试代理延迟，自动选择最优
- **支持**: Mihomo 等 HTTP/SOCKS5 代理

### 6. VPN 自动启动
- **文件**: [src/download/vpn_detector.py](src/download/vpn_detector.py)
- **功能**: 自动检测并启动 Mihomo 代理
- **特点**: 无需手动操作

## 🚀 使用方法

### 快速开始
```bash
# 方式 1: 双击启动（Windows）
启动GUI.bat

# 方式 2: 命令行启动
python launch_gui.py

# 方式 3: 系统测试
python test_system.py
```

### GUI 操作流程
1. **Elsevier 搜索** - 输入关键词，支持中文
2. **SciHub 下载** - 自动下载 PDF（Unpaywall 优先）
3. **处理临时文件** - 处理手动下载的 PDF
4. **文献分类** - 智能分类到不同类别
5. **MinerU 转换** - PDF 转 Markdown

## ⚙️ 配置说明

### API 密钥

```yaml
  email: "skdyoc@gmail.com"

# Elsevier API（用于文献搜索）

  api_key: "12c246d28f9c4eed838447b78644356a"

# GLM API（用于关键词翻译）

  api_keys:
    - "your_glm_api_key"
```

## 📌 重要提示

### SciHub 下载说明
1. **Unpaywall 优先** - 合法、免费，不需要 VPN
2. **SciHub 备用** - 需要 VPN，系统自动启动 Mihomo
3. **新文献 (2023-2026)** - SciHub 可能尚未收录
4. **旧文献 (2020-2022)** - 下载成功率更高

### 中文搜索
- 支持中文关键词输入
- 自动翻译为英文（需配置 GLM API）
- 或直接使用英文关键词

### 下载失败处理
1. 系统自动导出失败列表（NoteExpress 格式）
2. 手动下载 PDF 到 `pdfs/temp/`
3. 点击「处理临时文件」自动匹配
