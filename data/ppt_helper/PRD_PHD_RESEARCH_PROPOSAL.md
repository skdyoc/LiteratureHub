# PRD: 博士研究方向调研汇报 PPT 生成系统

## 1. 产品概述

### 1.1 产品名称
博士研究方向调研汇报 PPT 生成系统 - 大型风力机气动领域

### 1.2 应用场景
**用户身份**: 博一刚入学的研究生
**汇报对象**: 导师/研究团队
**汇报目标**:
- 展示对"大型风力机气动"领域的全面理解
- 找出研究空白和机会点
- 确定具体的研究分支和课题
- 制定可行的研究计划
- **说服导师同意研究方向**

### 1.3 核心价值主张
**从"我知道这个领域" → "我要研究这个" → "我准备这样研究"**

## 2. 需求背景

### 2.1 用户现状
- **学术阶段**: 博一新生，需要确定研究方向
- **数据基础**: 350 篇核心文献（2018-2026）的深度分析
- **分析成果**:
  - 8 大核心研究领域的深度分析
  - 5 大颠覆性创新
  - 研究热点与趋势
  - 技术发展脉络
  - 研究空白与挑战

### 2.2 汇报目标（D. 平衡三者）

**目标 A: 展示领域理解广度**
- 证明做了充分的文献调研
- 展示对领域的系统性认知
- 体现学术素养和研究能力

**目标 B: 发现研究机会**
- 识别研究空白和未解决问题
- 分析现有方法的局限性
- 找到可以突破的切入点

**目标 C: 提出研究方向**
- 基于研究空白，提出可行的研究方向
- 制定清晰的研究计划
- 展示研究的价值和可行性

### 2.3 汇报框架要求

按照博士开题汇报的标准结构：
1. **领域调研综述** - 展示理解
2. **研究空白分析** - 发现机会
3. **研究方向建议** - 提出方案
4. **研究计划制定** - 展示准备

## 3. 数据资产

### 3.1 信息层级架构

```
Level 0: PDF 文献（350 篇，2018-2026）
    ↓ MinerU 解析
Level 1: Markdown 全文
    ↓ Claude Code Agent 分析
Level 2: Agent 深度分析（8 个领域 × 5 个维度）
    ↓ 综合提炼
Level 3: Phase 精炼分析（phase1 + 8 个领域深度分析）
    ↓ 总结生成
Level 4: Part 综合分析（4 部分）
    ↓ 最终输出
Level 5: 15-20 页 HTML PPT 内容
```

### 3.2 核心分析结果

**八大核心研究领域**:
| 领域 | 文献数 | 研究成熟度 | 关键挑战 |
|-----|-------|----------|---------|
| 浮式海上风电气动弹性 | 78 | 热点上升 | 耦合机制复杂 |
| 气动外形优化 | 65 | 方法成熟 | AI 可解释性 |
| 尾流效应 | 52 | 快速发展 | 实时控制难 |

3. **中等复杂度高保真模型** - CFD-CSD 太贵，BEM 太简
4. **"块状"布局机理解析** - 效率提升 70%，机理未明
5. **AI 设计可解释性** - "黑箱"设计缺乏物理可信度

## 4. 功能需求

### 4.1 PPT 内容结构（15-20 页）

#### 第一部分：领域调研综述（4-5 页）- 展示理解

**Page 1: 封面与目录**
- 标题：大型风力机气动领域调研汇报
- 汇报人、时间、导师
- 汇报提纲

**Page 2: 研究背景与意义**
- 大型风力机发展趋势（20MW+）
- 气动设计的关键地位
- 调研范围与方法（350 篇文献，2018-2026）
- 汇报目标：确定研究方向

**Page 3: 八大核心研究领域全景**
- 4x2 网格展示
- 每个领域：名称、文献数、研究成熟度、核心问题
- 用颜色标注研究热度（上升/稳定/下降）

**Page 4: 技术发展脉络（2018-2026）**
- 三阶段时间线：
  - 阶段一（2018-2020）：传统方法成熟
  - 阶段二（2021-2023）：高保真方法突破
  - 阶段三（2024-2026）：智能化与综合优化
- 关键技术里程碑
- 发展趋势分析

**Page 5: 研究热点与高影响力工作**
- 5 大研究热点演变（频次+趋势）
- Top 5 高被引文献（标题+贡献+影响）
- 核心研究团队分布

#### 第二部分：研究空白分析（4-5 页）- 发现机会

**Page 6: 研究空白总览**
- 5 大研究空白列表
- 按影响程度排序
- 按解决难度分类
- 与研究热点的对应关系

**Page 7: 空白一：海洋湍流模型标准化**
- 问题：IEC 标准基于陆上，偏差 15-20%
- 影响：所有海上风电设计
- 现有工作：Syed-Mann 模型
- 研究机会：建立海洋-陆基修正系数，推动标准修订

**Page 8: 空白二：瞬态工况疲劳评估**
- 问题：启停/故障过程疲劳被忽视
- 现状：侧重稳态和极端工况
- 研究机会：建立瞬态疲劳评估标准
- 应用价值：提升设计安全性

**Page 9: 空白三：中等复杂度高保真模型**
- 问题：CFD-CSD 太贵（1秒需47秒CPU），BEM 太简
- 需求：平衡精度与效率
- 研究机会：降阶模型（ROM）工程化
- 预期：效率提升 10 倍，周期缩短 20%

**Page 10: 空白四与五**
- 空白四："块状"布局机理解析（效率提升70%，机理未明）
- 空白五：AI 设计可解释性（"黑箱"缺乏可信度）
- 对比分析：哪个更适合作为博士课题？

#### 第三部分：研究方向建议（3-4 页）- 提出方案

**Page 11: 研究方向筛选逻辑**
- 从 5 大空白 → 3 个候选方向
- 筛选标准：
  - 科学价值（创新性、影响力）
  - 可行性（数据、工具、时间）
  - 个人兴趣与匹配度
  - 导师研究方向契合度

**Page 12: 候选方向一：中等复杂度高保真模型**
- 研究内容：降阶模型（ROM）工程化
- 科学价值：填补方法空白，影响所有大型风机设计
- 可行性：理论成熟，有工具基础
- 预期成果：效率提升 10 倍，发表 2-3 篇高质量论文
- 风险与对策

**Page 13: 候选方向二：瞬态工况疲劳评估**
- 研究内容：瞬态载荷建模 + 疲劳累积方法
- 科学价值：填补设计标准空白，提升安全性
- 可行性：有实验数据支持，方法相对成熟
- 预期成果：建立评估标准，工业应用潜力大
- 风险与对策

**Page 14: 候选方向三：海洋湍流标准化**
- 研究内容：海洋湍流模型 + 修正系数 + 标准提案
- 科学价值：修正 15-20% 系统偏差，影响深远
- 可行性：需长期现场数据，周期较长
- 预期成果：标准修订提案，高影响力
- 风险与对策

**Page 15: 方向对比与推荐**
- 三维度对比表：科学价值 / 可行性 / 时间周期
- 推荐方向（根据个人情况）
- 推荐理由

#### 第四部分：研究计划制定（3-4 页）- 展示准备

**Page 16: 推荐方向：中等复杂度高保真模型**
- 为什么选择这个方向？
- 与导师研究方向的契合度
- 个人基础与兴趣
- 研究的独特性

**Page 17: 研究内容与目标**
- **研究目标**：
  - 短期（1 年）：建立 ROM 框架，验证可行性
  - 中期（2-3 年）：工程化实现，精度验证
  - 长期（4-5 年）：形成设计工具，发表成果
- **研究内容**：
  - 文献调研与现状分析（已完成 ✅）
  - ROM 方法研究与实现
  - 多保真度数据融合
  - 精度验证与案例分析
  - 工具开发与应用

**Page 18: 技术路线与方法**
- 技术路线图（流程图）
- 方法选择：
  - 理论研究：系统辨识、POD、神经网络
  - 数值验证：CFD-CSD 高保真模拟
  - 实验对标：公开数据集、FINO1 平台
- 工具与资源：OpenFOAM、Python、PyTorch

**Page 19: 研究计划时间表**
- **博一（第 1 年）**：课程学习 + 文献深入 + ROM 框架搭建
- **博二（第 2 年）**：方法实现 + 精度验证 + 发表第 1 篇论文
- **博三（第 3 年）**：工程化优化 + 案例应用 + 发表第 2 篇论文
- **博四（第 4 年）**：工具开发 + 博士论文撰写
- **博五（第 5 年）**：论文完善 + 答辩准备

**Page 20: 总结与展望**
- **调研总结**：完成了 350 篇文献的系统性分析
- **研究空白**：识别了 5 大关键空白
- **研究方向**：推荐"中等复杂度高保真模型"
- **研究价值**：科学价值 + 工程价值 + 个人发展
